## Bluesat Juno Vibration Module

The current directory contains the code for the UNSW Rocketry vibration module. The components used for this project are:
- A NUCLEO F042K6
- A microSD card connected via SPI
- An IMU module connected via I2C

Documentation is located at:
<https://bluesat.atlassian.net/wiki/spaces/SP/pages/3017867282/Vibration+Module>
